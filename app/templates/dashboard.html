How the Pipeline Works

1
OCR (Text Extraction)

EasyOCR reads text from images/PDFs. Falls back to Tesseract if needed.

2
ML Classification

TF-IDF + Logistic Regression identifies if it's an invoice, receipt, or bank statement.

3
NER Field Extraction

Regex patterns + BERT NER model extract dates, amounts, vendor names, etc.

4
LLM Correction (GPT + RAG)

GPT-4o-mini fixes errors and normalizes formats. Uses FAISS vector search to find similar past documents as examples.

5
Excel Export

pandas converts the extracted JSON to a formatted .xlsx file with styled headers.
